# Math Quest 7B — v0.2

Versión integrada del juego de Math para Santiago.

## Novedades de v0.2
- Pantalla de inicio tipo videojuego.
- Mapa visual de los seis mundos.
- Estructura organizada en carpetas.
- Perfil con XP, monedas, cristales y racha.
- Lesson 5.9 funcional.
- Lesson 6.1 Speed Raceway funcional.
- Academia, práctica, batalla/carrera, generador, jefe final y examen.
- Diario, logros, modo mamá/papá y ajustes.
- Efectos de sonido generados por el navegador.
- Guardado automático en LocalStorage.

## Ejecutar
Abra `index.html` con Chrome, Edge o Firefox.

## Subir a GitHub
Reemplace los archivos anteriores por:
- `index.html`
- carpeta `css`
- carpeta `js`
- carpeta `data`
- `README.md`
